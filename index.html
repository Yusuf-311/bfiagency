Agency Value Proposition

Merupakan bentuk bisnis kemitraan yang bertujuan untuk mendapatkan calon-calon nasabah BFI Finance yang ingin mengajukan pembiayaan berdasarkan jaminan (collateral) kendaraan bermotor, atau yang biasa disebut juga dengan istilah Refinancing/Non Dealer Financing (NDF). Untuk Agent Mobil, konteks jaminan pembiayaan konsumen dalam hal ini yaitu jaminan kendaraan jenis mobil, pada buku saku agent ini akan membantu Anda untuk memahami secara umum mengenai proses bisnis sebagai Agent BFI produk mobil dan beragam benefit yang akan Anda dapatkan.

CEPAT

  • Proses pengajuan sampai dengan persetujuan cepat
  • Respon pelayanan terhadap agent yang cepat
  • Proses pemberian bonus cepat

TRANSPARAN

  • Proses transfer bonus langsung oleh BFI Finance
  • Agent mendapatkan slip gaji bonus
  • Agent dapat mengetahui setiap unit kendaraan masuk

DUKUNGAN

  • Pendampingan personal untuk membantu kebutuhan masing-masing agent
  • Aktivitas Agent didukung dengan tools
  • Penjualan yang didukung oleh sistem
